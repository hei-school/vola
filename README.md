# Vola

Make your users pay as per usual,
then retrieve their payments programmatically.

_Vola_ is the malagasy word for money.

---

API definition can be browsed [here](https://petstore.swagger.io/?url=https://42cwka3n4ifcp7ufheyrpmph240iuaxo.lambda-url.eu-west-3.on.aws/v3/api-docs).

Currently supported Payment Service Provider (PSP) are:
- Orange Money Madagascar
